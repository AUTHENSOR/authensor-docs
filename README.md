# Authensor Docs

[![CI](https://github.com/AUTHENSOR/authensor-docs/actions/workflows/ci.yml/badge.svg)](https://github.com/AUTHENSOR/authensor-docs/actions/workflows/ci.yml)

Documentation, compliance guides, and standards for **Authensor** — the open-source safety stack for AI agents.

Authensor provides action authorization, approval workflows, content safety scanning, real-time monitoring, and cryptographic audit trails for tool-using AI agents.
